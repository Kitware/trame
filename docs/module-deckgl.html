Deck.gl

deck.gl is a WebGL-powered framework for visual exploratory data analysis of large datasets which has a Python layer available under pydeck.

trame leverages pydeck (see documentation and examples here) to build data layers which deck.gl then renders. Developers can make interactive data layers on top of base map providers like Mapbox.
