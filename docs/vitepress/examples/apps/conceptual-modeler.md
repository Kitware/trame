# ParFlow Conceptual Modeler

The ParFlow Conceptual Modeler is a Python/trame application for building complex 3D geological models from interface and orientation data. It relies on [GemPy](https://www.gempy.org/) to implicitly generate subsurface geology, letting geoscientists explore layered earth structures interactively in the browser instead of hand-building every surface.
