# Trame + Catalyst

trame-catalyst is a trame application for live-monitoring a running simulation connected through [ParaView Catalyst](https://docs.paraview.org/en/latest/Catalyst/), an in-situ analysis framework that avoids the I/O bottleneck of writing every time step to disk. Since it's a web app using remote rendering, you can watch the simulation from any device, with no client-side installation and no need for a powerful local GPU. See the [announcement blog post](/blogs/monitor-your-simulation-in-your-web-browser-with-trame-and-catalyst) for a walkthrough.
