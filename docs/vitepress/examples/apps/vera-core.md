# VERA Core

VERA Core is a visualization application built with VTK and trame for exploring output from VERA (Virtual Environment for Reactor Applications), a multi-physics nuclear reactor simulation framework. It provides instantaneous 2D and 3D views, 1D plots, and alphanumeric data so engineers can quickly interpret large-scale reactor analysis results through a web-based interface.
