# XAITK Saliency Web Demo

This trame application demonstrates [XAITK Saliency](https://xaitk-saliency.readthedocs.io/), a library of explainable AI (XAI) techniques for visualizing model behavior. It lets you apply saliency methods such as RISE and sliding window to image classification, object detection, and similarity tasks, and can be run locally, in Jupyter, or in the cloud.
